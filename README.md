# Shieldxx Home Assistant Add-ons

Home Assistant add-on repository.

## Add-ons

- **Beszel Hub** — lightweight self-hosted server monitoring hub ([beszel.dev](https://beszel.dev)). Runs the official `henrygd/beszel` image as a native HAOS add-on. Web UI on port `8090`, data persisted across restarts/updates.

## Install

In Home Assistant: **Settings → Add-ons → Add-on Store → ⋮ (top right) → Repositories**, then add:

```
https://github.com/Shieldxx/hassio-beszel-hub
```

Refresh the store and install **Beszel Hub** from the *Shieldxx Home Assistant Add-ons* section.
